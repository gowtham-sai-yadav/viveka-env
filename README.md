# Viveka

> **Viveka** (विवेक, Sanskrit): *"the wisdom to discriminate."*

An OpenEnv reinforcement-learning environment that trains LLM agents to do three things on India's Digital Public Infrastructure (UPI, DigiLocker, IRCTC):

1. **Predict whether an action is reversible *before* executing it.**
2. **Emit a calibrated confidence score on every action.**
3. **Ask the user when uncertain or facing an irreversible decision** — instead of guessing.
